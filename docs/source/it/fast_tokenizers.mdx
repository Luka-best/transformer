# Usare i tokenizzatori di ðŸ¤— Tokenizers

Il tokenizzatore [`PreTrainedTokenizerFast`] dipende dalla libreria [ðŸ¤— Tokenizers](https://huggingface.co/docs/tokenizers). I tokenizzatori 
disponibili nella libreria ðŸ¤— Tokenizers possono essere caricati facilmente con ðŸ¤— Transformers.

Prima di entrare nei dettagli, iniziamo creando un tokenizzatore con poche righe di codice:

```python
>>> from tokenizers import Tokenizer
>>> from tokenizers.models import BPE
>>> from tokenizers.trainers import BpeTrainer
>>> from tokenizers.pre_tokenizers import Whitespace

>>> tokenizer = Tokenizer(BPE(unk_token="[UNK]"))
>>> trainer = BpeTrainer(special_tokens=["[UNK]", "[CLS]", "[SEP]", "[PAD]", "[MASK]"])

>>> tokenizer.pre_tokenizer = Whitespace()
>>> files = [...]
>>> tokenizer.train(files, trainer)
```

Adesso abbiamo un tokenizzatore addestrato sui file che abbiamo definito. Da qui possiamo sia continuare a usarlo in quel runtume o salvarlo in un file JSON per riutilizzarlo in seguito.


## Caricare un tokenizzatore da un oggetto

Vediamo ora come utilizzare questo oggetto con la libreria ðŸ¤— Transformers. La classe
[`PreTrainedTokenizerFast`] puÃ² essere istanziata in modo semplice, accettando il tokenizzatore che abbiamo definito
come argomento:

```python

>>> fast_tokenizer = PreTrainedTokenizerFast(tokenizer_object=tokenizer)
```

## Caricare da un file JSON

Per caricare un tokenizzatore da un file JSON, iniziamo salvando il nostro tokenizzatore:

```python
>>> tokenizer.save("tokenizer.json")
```

Il percorso in cui abbiamo salvato il file puÃ² essere passato al metodo di inizializzazione di [`PreTrainedTokenizerFast`] utilizzando il
parametro `tokenizer_file`:

```python

>>> fast_tokenizer = PreTrainedTokenizerFast(tokenizer_file="tokenizer.json")
```
